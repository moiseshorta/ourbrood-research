# TiMem: Temporal-Hierarchical Memory Consolidation for Long-Horizon Conversational Agents

**Authors:** Kai Li, Xuanqing Yu, Ziyi Ni, Yi Zeng, Yao Xu, Zheqing Zhang, Xin Li, Jitao Sang, Xiaogang Duan, Xuelei Wang, Chengbao Liu, Jie Tan  
**Link:** https://arxiv.org/abs/2501.11345 (January 2026)  
**Submitted:** January 2026

## Abstract

Long-horizon conversational agents face the challenge of maintaining coherent, context-aware interactions over extended periods. TiMem proposes temporal-hierarchical memory consolidation inspired by human cognitive memory processes...

## Relevance to OurBrood

This paper addresses **temporal memory organization** - critical for Mother's role in long exhibition sessions:

1. **Temporal Organization**: Mother interacts with many visitors across hours/days. TiMem addresses how to organize memories by time.

2. **Consolidation Process**: The paper's consolidation mechanism parallels Mother's 90s recap system but adds temporal structure.

3. **Long-Horizon Coherence**: Exhibition sessions span hours. TiMem provides patterns for maintaining coherence.
