# AdaMARP: An Adaptive Multi-Agent Interaction Framework for General Immersive Role-Playing

**Authors:** Zhenhua Xu, Dongsheng Chen, Shuo Wang, Jian Li, Chengjie Wang, Meng Han, Yabiao Wang

**Source:** arXiv, January 2026

**Link:** https://arxiv.org/abs/2501.XXXXX (verify)

---

## Summary

This paper introduces AdaMARP (Adaptive Multi-Agent Role-Playing), a framework for creating immersive role-playing experiences using Large Language Models. The system addresses the challenge of maintaining character consistency and narrative coherence while allowing for adaptive, emergent storytelling.

---

## Key Contributions

### 1. Multi-Agent Architecture
- Multiple LLM agents each embody distinct characters
- Coordination framework that maintains narrative coherence
- Adaptive response to player/participant actions

### 2. Immersive Role-Playing Design
- Focus on immersion and character embodiment
- Balance between predetermined narrative structure and emergent possibilities
- Real-time adaptation to participant choices

### 3. General Framework
- Not limited to specific genres or settings
- Applicable across different role-playing contexts
- Scalable from solo to group experiences
